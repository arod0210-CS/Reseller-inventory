# 📦 Reseller Inventory App

🚀 Live App: https://arod0210-CS.github.io/Reseller-inventory/

---

## 🧠 Overview

The **Reseller Inventory App** is a lightweight, browser-based tool designed to help resellers manage and track their inventory efficiently.  

Built with simplicity and speed in mind, this app allows users to organize items, track financials, and manage product data without needing a backend or database setup.

---

## ✨ Features

- 📋 Add, edit, and delete inventory items  
- 💰 Track cost, price, and profit  
- 🏷️ Labeling and categorization system  
- 🌍 Multi-language support (i18n ready)  
- 📊 Basic analytics tracking  
- 📷 Scanner-ready structure (future expansion)  
- 💾 Local storage (no login required)  
- 📱 Responsive design (desktop & mobile)

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- LocalStorage API
